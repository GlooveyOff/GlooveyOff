## 🎯 Сейчас в разработке — ОНВИЛЛ

Сервис промокодов, который сам подбирает купон под то, что человек смотрит
в интернет-магазине, вместо того чтобы искать промокод руками.

**Как это работает:**

- браузерное расширение следит за открытой вкладкой магазина
- по хлебным крошкам, заголовку страницы и категории из URL определяется,
  что именно смотрит пользователь
- ключевые слова сопоставляются с описаниями активных промокодов —
  подбирается самый релевантный по теме
- если тематического промокода нет — показывается лучший общий промокод
  магазина с пометкой
- работает live на SPA-сайтах (переход между страницами без перезагрузки)
- код промокода копируется в буфер одним кликом

**Поддерживаемые магазины:** Ozon, Wildberries, Яндекс.Маркет, AliExpress,
Lamoda, DNS, Мегамаркет, Ситилинк, М.Видео + общая эвристика для остальных

**Из чего состоит:**

| Репозиторий | Роль |
|---|---|
| [`onwill-extension`](https://github.com/GlooveyOff/onwill-extension) | расширение для Chrome/Firefox (Manifest V3) |
| `onwill-backend` | API `/api/stores`, отдаёт актуальные промокоды |
| `onwill-site` | сайт [onwill.ru](https://onwill.ru) |

Статус: 🚧 активная разработка, текущая версия расширения — **1.2.0**
